📦 Projeto 1 - Análise de Indicadores Logísticos

Dashboard interativo desenvolvido em Power BI para análise de indicadores logísticos, com foco em controle de estoque, giro, cobertura e ruptura de produtos.

Principais análises:

  • Monitoramento de giro de estoque
  • Análise de cobertura atual dos produtos
  • Identificação de itens em ruptura
  • Visão de itens com baixa cobertura e excesso

🚛 Projeto 2 - Análise de Transporte Logístico

Dashboard desenvolvido para monitoramento de performance de transporte, com foco em custos, eficiência operacional e apoio à tomada de decisão.

Principais análises:

  • Análise de custo por operação
  • Monitoramento da eficiência logística
  • Identificação de gargalos no transporte
  • Acompanhamento de indicadores de desempenho

Insights gerados:

  • Identificação de operações com maior impacto em custo
  • Visão de oportunidades de otimização logística
  • Suporte à tomada de decisão com base em indicadores

🧠 Projeto 3 - Automação e Análise de Dados

Projeto voltado para automação e análise de dados aplicados à logística, utilizando Python para geração de alertas, apoio operacional e identificação de padrões críticos.

Funcionalidades:

  • Análise automatizada de estoque
  • Identificação de produtos críticos
  • Geração de alertas logísticos
  • Suporte à tomada de decisão
